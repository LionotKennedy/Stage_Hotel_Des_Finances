Pour récupérer les données de vos dossiers et les afficher dans la console, vous devez d'abord utiliser
 la fonction `useGetFolders` que vous avez définie avec `react-query`. Voici comment vous pouvez intégrer 
 cela dans votre composant `TableResponsive` :

### Étapes à Suivre

1. **Importez `useGetFolders` dans votre composant**.
2. **Utilisez le hook pour récupérer les données**.
3. **Affichez les données dans la console** lorsque les données sont disponibles.

### Mise à Jour du Composant `TableResponsive`

Voici comment vous pouvez modifier le composant `TableResponsive` pour récupérer les dossiers et les afficher dans la console :

```javascript
import React, { useEffect, useRef, useState } from 'react';
import search from "../../assets/image/search.png"
import { MdEdit, MdDelete, MdVisibility, MdAdd } from 'react-icons/md';
import { FaArrowUp } from 'react-icons/fa';
import { AnimatePresence } from 'framer-motion';
import { useGetFolders } from '../../services/folderService'; // Assurez-vous que le chemin est correct
import "./tableResponsive.scss"
import CustomModal from '../MUI/CustomModal';

const TableResponsive = () => {
    const tableRef = useRef(null);
    const searchRef = useRef(null);
    const [searchType, setSearchType] = useState('name');
    const [modalOpen, setModalOpen] = useState(false);

    // Utilisez le hook pour récupérer les dossiers
    const { data: folders, isLoading, isError } = useGetFolders();

    const handleOpenModal = () => setModalOpen(true);
    const handleCloseModal = () => setModalOpen(false);

    useEffect(() => {
        if (folders) {
            console.log('Données des dossiers:', folders); // Affichez les données dans la console
        }
    }, [folders]);

    // Afficher un message d'erreur ou un indicateur de chargement
    if (isLoading) return <div>Loading...</div>;
    if (isError) return <div>Error loading folders</div>;

    return (
        <div className='container__table'>
            <main className="table" ref={tableRef} id="customers_table">
                <section className="table__header">
                    <select className='searchByeverything' value={searchType} onChange={(e) => setSearchType(e.target.value)}>
                        <option value="name">Search by Name</option>
                        <option value="id">Search by ID</option>
                    </select>
                    <div className="input-group">
                        <input type="search" placeholder="Search Data..." ref={searchRef} />
                        <img src={search} alt="" />
                    </div>
                    <div className='option_right'>
                        <MdAdd onClick={handleOpenModal} className="icon_add" style={{ marginLeft: '10px', fontSize: '24px' }} />
                    </div>
                </section>
                <section className="table__body">
                    <table className='table'>
                        <thead className='thead'>
                            <tr>
                                <th className='th'>Id <span className="icon-arrow"><FaArrowUp /></span></th>
                                <th className='th'>Name <span className="icon-arrow"><FaArrowUp /></span></th>
                                {/* Ajoutez d'autres colonnes si nécessaire */}
                                <th className='th'>Actions <span className="icon-arrow"><FaArrowUp /></span></th>
                            </tr>
                        </thead>
                        <tbody className='tbody'>
                            {folders && folders.map((folder) => (
                                <tr key={folder.id}> {/* Remplacez folder.id par l'ID correct si nécessaire */}
                                    <td className="td">{folder.id}</td>
                                    <td className="td">{folder.name}</td>
                                    <td className="td">
                                        <MdEdit className="action-icon icon" title="Update" />
                                        <MdDelete className="action-icon icon" title="Delete" />
                                        <MdVisibility className="action-icon icon" title="Read" />
                                    </td>
                                </tr>
                            ))}
                        </tbody>
                    </table>
                </section>
                <AnimatePresence>
                    <CustomModal open={modalOpen} handleClose={handleCloseModal} />
                </AnimatePresence>
            </main>
        </div>
    );
}

export default TableResponsive;
```

### Explications des Changements

1. **Importation de `useGetFolders`** : J'ai ajouté l'importation de votre hook `useGetFolders` au début du fichier.
   
2. **Utilisation du hook** : J'ai utilisé `const { data: folders, isLoading, isError } = useGetFolders();`
 pour récupérer les dossiers et gérer les états de chargement et d'erreur.

3. **Affichage des dossiers** : J'ai ajouté un `console.log` dans un `useEffect` qui affiche les données des dossiers 
chaque fois qu'elles sont récupérées.

4. **Gestion des états de chargement et d'erreur** : J'ai inclus une gestion simple des états de chargement et d'erreur
 pour informer l'utilisateur.

5. **Affichage des données dans le tableau** : Les données des dossiers sont maintenant affichées dans le tableau sous 
les colonnes appropriées. Assurez-vous de remplacer `folder.id` et `folder.name` par les bonnes propriétés selon votre API.

### Test

Une fois que vous avez apporté ces modifications, vous devriez voir les données de dossiers dans la console lorsque 
le composant est monté et que les données sont récupérées avec succès. Assurez-vous que votre API fonctionne et que 
les données sont correctement retournées.
