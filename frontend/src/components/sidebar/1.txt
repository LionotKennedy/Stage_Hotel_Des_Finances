Bien sûr, je comprends que vous souhaitez récupérer le rôle de l'utilisateur depuis la base de données plutôt que de le stocker dans le localStorage. Voici comment vous pouvez modifier votre code pour implémenter cette logique :

### Modification du composant Login

Dans le composant Login, assurez-vous de récupérer le rôle de l'utilisateur et de l'envoyer au composant App :

```jsx
const Login = ({ onLogin }) => {
  // ... autres codes inchangés ...

  const handleLogin = async () => {
    // ... code existant ...
    
    if (result.success) {
      const userStatus = result.data.status;
      const userId = result.data._id;
      const userRole = result.data.role; // Récupère le rôle de l'utilisateur

      console.log('Utilisateur connecté:', {
        id: userId,
        status: userStatus,
        role: userRole
      });

      // Stocker les informations de l'utilisateur dans le localStorage
      localStorage.setItem('userId', userId);
      localStorage.setItem('userStatus', userStatus);
      localStorage.setItem('userRole', userRole);

      const userData = await getProfile(userId, result.accessToken);
      console.log("Données utilisateur après connexion:", userData);
      onLogin(userData);
      onLogin(result.data);
      console.log("Données utilisateur stockées :", result.data);
      console.log("Id :", userId);
    } else {
      // ... code existant ...
    }
  };

  // ... reste du code inchangé ...
};
```

### Modification du composant App

Dans le fichier App.jsx, modifiez-le pour récupérer et utiliser le rôle de l'utilisateur :

```jsx
import React, { useState, useEffect } from 'react';
import { BrowserRouter as Router, Route, Routes, Navigate } from 'react-router-dom';
import Login from './pages/login/Login';
import Layout from './layout/Layout';
import Dossier from './pages/dossier/Dossier';
import Home from './pages/home/Home';
import Visa from './pages/visa/Visa';
import Archive from './pages/archive/Archive';
import Journal from './pages/journal/Journal';
import User from './pages/user/User';
import Profile from './pages/profile/Profile';
import ArchiveMore from "./pages/archivemore/ArchiveMore";
import ProtectedRoute from './ProtectedRoute'; 
import { getProfile } from './services/authServices'; 
import UserPage from "./pages/userpage/UserPage";
import ShowUser from "./pages/showuser/ShowUser";

function App() {
  const [isAuthenticated, setIsAuthenticated] = useState(false);
  const [token, setToken] = useState(null);
  const [userRole, setUserRole] = useState(null);

  useEffect(() => {
    const storedUserId = localStorage.getItem('userId');
    if (storedUserId) {
      setToken(null); // Supprimer le token existant
      setIsAuthenticated(true);
      const storedUserRole = localStorage.getItem('userRole');
      setUserRole(storedUserRole);
      
      const userId = storedUserId;
      getProfile(userId, null).then(userData => {
        console.log("Données utilisateur après connexion:", userData);
      });
    }
  }, []);

  const handleLogout = () => {
    localStorage.removeItem('userId');
    localStorage.removeItem('userStatus');
    localStorage.removeItem('userRole');
    setIsAuthenticated(false);
  };

  useEffect(() => {
    const intervalId = setInterval(async () => {
      const storedUserId = localStorage.getItem('userId');
      if (storedUserId) {
        try {
          await getProfile(storedUserId, null); // Essayez de récupérer le profil
          console.log("Token valide");
        } catch (error) {
          console.error("Erreur lors de la récupération du profil:", error);
          handleLogout(); // Déconnexion en cas d'autres erreurs
        }
      }
    }, 60000); // Vérifier toutes les 1 minute

    return () => clearInterval(intervalId);
  }, []);

  return (
    <Router>
      <Routes>
        <Route
          path="/"
          element={isAuthenticated ? <Navigate to="/accueil" /> : <Login onLogin={() => setIsAuthenticated(true)} />}
        />
        {isAuthenticated && (
          <Route element={<Layout onLogout={handleLogout} />}>
            <Route path="/accueil" element={<Home />} />
            <Route path="/courrier" element={<Dossier />} />
            <Route path="/visa" element={<Visa />} />
            <Route path="/archive" element={<Archive />} />
            <Route path="/journal" element={
              userRole === '0' || userRole === null ? <Navigate to="/accueil" /> : <Journal />
            }/>
            <Route path="/utilisateur" element={
              userRole === '0' || userRole === null ? <Navigate to="/accueil" /> : <User />
            }/>
            {/* Autres routes conditionnelles basées sur le rôle */}
          </Route>
        )}
        <Route path="*" element={<Navigate to="/" />} />
      </Routes>
    </Router>
  );
}

export default App;
```

### Modification du composant SideBar

Dans le fichier Sidebar.jsx, modifiez-le pour utiliser le rôle de l'utilisateur :

```jsx
import React from 'react';
import { Link, useLocation } from 'react-router-dom';
import "./sidebar.scss";
import sidebar_items from '../../Data/Sidebar__route.json';
import logo from '../../assets/images/logo.png';

const SidebarItem = props => {
  const active = props.active ? 'active' : '';
  
  return (
    <div className="sidebar__item">
      <div className={`sidebar__item-inner ${active}`}>
        <i className={props.icon}></i>
        <span>
          {props.title}
        </span>
      </div>
    </div>
  )
}

const SideBar = () => {
  const location = useLocation();

  // Fonction pour filtrer les éléments du menu basé sur le rôle
  const filteredItems = sidebar_items.filter(item => {
    if (localStorage.getItem('userRole') === '0') {
      return item.route !== '/journal' && item.route !== '/utilisateur';
    }
    return true;
  });

  const activeItem = filteredItems.findIndex(item => 
    location.pathname.startsWith(item.route)
  );
  
  return (
    <div className='sidebar'>
      <div className="sidebar__logo">
        <img className='logo_img' src={logo} alt="company logo" />
        {/* Bro-Coding */}
      </div>
      {filteredItems.map((item, index) => (
        <Link to={item.route} key={index}>
          <SidebarItem
            title={item.display_name}
            icon={item.icon}
            active={index === activeItem}
          />
        </Link>
      ))}
    </div>
  )
}

export default SideBar;
```

### Explications des modifications

1. Dans le composant Login, nous récupérons maintenant le rôle de l'utilisateur et le stockons dans localStorage.
2. Dans le composant App, nous ajoutons un nouvel état `userRole` pour stocker le rôle de l'utilisateur.
3. Nous modifions les routes conditionnelles dans App.jsx pour utiliser le rôle de l'utilisateur pour décider si afficher ou masquer certaines pages.
4. Dans le composant SideBar, nous utilisons toujours localStorage pour récupérer le rôle de l'utilisateur, car c'est la méthode que vous préférez.

Cette approche permet de récupérer le rôle de l'utilisateur depuis la base de données au moment du login et d'utiliser ce rôle pour contrôler l'accès aux différentes pages et menus dans votre application. Assurez-vous que votre API retourne toujours le rôle de l'utilisateur avec les informations de connexion.
