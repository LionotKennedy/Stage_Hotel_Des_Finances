Oui, il est tout à fait possible d'utiliser des variables SCSS pour définir un dégradé linéaire et de les appliquer à différents éléments dans un composant React. Voici comment procéder :

### Étape 1: Créer une variable SCSS

Dans votre fichier SCSS (par exemple, `styles.scss`), vous pouvez définir une variable contenant votre dégradé linéaire :

```scss
// styles.scss

$linear-gradient: linear-gradient(90deg, #ff7e5f, #feb47b); // Exemple de dégradé

// Utilisation de la variable pour le background
.component {
  background: $linear-gradient;
}

// Utilisation de la variable pour d'autres éléments, par exemple un bouton
.button {
  background: $linear-gradient;
  border: none;
  color: white;
  padding: 10px 20px;
  cursor: pointer;

  &:hover {
    opacity: 0.8; // Opacité au survol
  }
}

// Utilisation de la variable pour le texte
.text {
  background: $linear-gradient;
  -webkit-background-clip: text;
  -webkit-text-fill-color: transparent; // Pour le texte en dégradé
}
```

### Étape 2: Appliquer le style dans votre composant React

Assurez-vous d'importer le fichier SCSS dans votre composant React :

```jsx
import React from 'react';
import './styles.scss';

export default function MyComponent() {
  return (
    <div className="component">
      <h1 className="text">Titre avec dégradé</h1>
      <button className="button">Cliquez ici</button>
    </div>
  );
}
```

### Étape 3: Résultat

Avec ces étapes, vous aurez un composant avec un fond dégradé, un bouton utilisant le même dégradé comme couleur de fond et du texte en dégradé.

### Remarques

1. **Compatibilité du navigateur** : Assurez-vous que le dégradé fonctionne dans les navigateurs cibles. Les préfixes comme `-webkit-` sont souvent nécessaires pour une compatibilité maximale.
2. **Responsivité** : Vous pouvez ajuster les couleurs et les directions du dégradé selon les besoins de votre design.
3. **Modularité** : En utilisant des variables SCSS, vous pouvez facilement modifier le dégradé à un seul endroit, ce qui affectera tous les composants qui l'utilisent.
